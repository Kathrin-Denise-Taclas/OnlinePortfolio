Academic Outputs 

UNDERGRADUATE THESIS

The main objective of the study, Banana Variety Classification Using ResNet50 Convolutional Neural Network on Android Smartphones, was to develop an android-based application that can classify seven banana varieties. With Python, the ResNet50 CNN model was developed and evaluated with its classification accuracy, multi-class logarithmic loss, and turnaround time. The application was built through Android Studio using the Java language mainly.

The paper was selected as one of the competing research projects in the Bicol University College of Science Student Forum 2022 under the Development Category for the theme “Scientific, Technological and Techno-Entrepreneurship in Industry, Energy, Emerging Technologies for Global Competitiveness”.
